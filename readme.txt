3.All data are stored in a parent table, where you can find a property column to differ subclasses from each other. For mapping this column, we should use a @DiscriminatorColumn(name = "column_name") annotation with parent class. As for child classes, there is a special annotation, called @DiscriminatorValue, where we should choose a property value for each subclass. 

3.Все данные хранятся в родительской таблице, разделяются проперти, которая хранится в аннотации над родителем - @DiscriminatorColumn(name = "column_name"). Над потомками вешается @DiscriminatorValue аннотация, в которой указывается значения столбца @DiscriminatorColumn. Всего одна таблица.
